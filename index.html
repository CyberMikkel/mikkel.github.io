Om mig

Jeg er ved at lære at lave hjemmesider med HTML, CSS og JavaScript.

Du er meget tæt på — din dark mode virker allerede næsten 😄 Problemet er bare, at din body har et fast background-image, så .dark ændrer ikke baggrunden synligt. Du kan gøre det sådan her i stedet: /* 🌙 Dark mode */ .dark { background-color: #121212; color: white; /* skift baggrundsbillede */ background-image: url("dark-baggrund.jpg"); } .dark nav a { color: #00ffff; } .dark h1 { color: #ffcc00; } .dark button { background: linear-gradient(135deg, #333, #666); } .dark img { box-shadow: 0 6px 15px rgba(255,255,255,0.1); } Og så skal du rette en lille fejl i din CSS øverst: Du har: body { ... } } Der er en ekstra } bagefter. Det skal være: body { font-family: Arial, sans-serif; background-image: url("skærmbillede"); background-size: cover; background-position: center; background-repeat: no-repeat; text-align: center; padding: 40px; margin: 0; transition: 1s; color: white; } Du kan også gøre darkmode endnu federe med forskellige farver: .dark { background-color: black; color: #00ffcc; } Eller: .dark { background: linear-gradient(to bottom, #000000, #222244); } Din JavaScript til darkmode er allerede korrekt: function toggleDarkMode() { document.body.classList.toggle("dark"); } Hvis du vil, kan jeg også hjælpe dig med: automatisk gemt darkmode smooth animation neon/cyberpunk tema rigtig lys-mode + mørk-mode skifte ikon 🌙☀️ få GitHub Pages til at vise billeder korrekt giv mig den fulde kode så jeg kan copy paste den ind Min side
